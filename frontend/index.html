🔥 Gas Tracker

What is this?

Gas Tracker calculates your total gas spending across multiple EVM chains and gives you a fun verdict on how much you've burned. Perfect for checking if you're a "Grass Toucher" or "Gas Royalty".

API Endpoints $0.05 per request

GET /v1/gas-spent/:address
POST /v1/gas-spent
Body: { "address": "0x...", "chains": ["ethereum", "base"] }
